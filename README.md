# rps-bot
A rock-paper-scissors bot that learns a best response to a fixed opponent strategy using Counterfactual Regret Minimization: https://wwwf.imperial.ac.uk/~dturaev/neller-lanctot.pdf)

#### Run
	ruby app.rb
